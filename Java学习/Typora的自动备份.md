# Typora Helper

## version 1.0实现了如下功能,目前只有Linux版本:

1. 定时的检测是否有更新(修改或者新增)的文件(现阶段写死为"md文件"),对新增修改文件中的本地图片链接进行图片的备份(现阶段只能要上传到Github的图片仓库上)并替换为github的图片地址,然后将其上传到笔记仓库上.

## 如何使用我们的程序

1. 在Github上新建一个笔记仓库和图片仓库
2. 设置全局用户,密码
3. 执行代码
